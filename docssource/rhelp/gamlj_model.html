gamlj_modelR Documentation

Extract the R model object contained in the GAMLj results

Description

Return the estimated model object of the class lm, lmer, glm depending on the model.

Usage

gamlj_model(object)

Arguments

object

a gamlj results object of the class 'gamlj*Results'

Value

a R object from lm(), glm(), lmer() or glmer()

Author(s)

Marcello Gallucci

Examples

data('qsport')
obj<-gamlj::gamljGlm(
   formula = performance ~ hours,
   data = qsport)
 model<-gamlj_model(obj)
